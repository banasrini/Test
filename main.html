Basic Introduction

Explore the Embedded Devices

Choose from a list of embedded devices supported by Pubnub for this dashboard. Here we will demonstrate connecting a Raspberry Pi and an Arduino to the Dashboard. The following are the features provided by this dashboard.

Send

Send() lets you connect to a channel, and to publish messages to every other device listening on that channel

Connect

Connect() allows you to listen to other devices, be it an Arduino board, a Raspberry Pi or any other embedded device supported by Pubnub.

Who's online

Whats the point of sending and connecting if you can't see who's online? This simple dashboard lets you check the different devices online at any instant.

Raspberry Pi

This post takes you through the process of connecting the Raspberry Pi to the PubNub network and communicating to other devices via the PubCon.

The Raspberry Pi is a credit-card sized computer that plugs into your TV and a keyboard. It is a capable little computer which can be used in electronics projects, and for many of the things that your desktop PC does, like spreadsheets, word-processing and games. Check out their website at Raspberry Pi to set up your Pi.

PubNub provides extensive documentation and support for the Raspberry Pi here . Once you are up and running with the Pi, feel free to choose your favorite language and start connecting. I have used Python for this project.

View details Â»

STEP 1: Connect the Pi to a monitor, keyboard, mouse and ethernet cable. Login in to the Raspberry Pi OS and open IDLE, the python IDE.

STEP 2: Open a new window and paste the following code. 

from Pubnub import Pubnub
from IoT_wrapper import iotwrapper

pi = iotwrapper(publish_key = 'demo', 
		subscribe_key = 'demo', 
		uuid = 'PI')

channel = 'button-reply'
message = "hello from pi"

def callbackfn(message):
  print(message)
  
pi.send(channel, message)
		
pi.connect(channel, callbackfn)

The above code lets you set up the following :

  • publish key using "publish_key"
  • subscribe key using "subscribe_key"
  • UUID using "uuid"
  • channel you wish to subscribe and publish to using "channel"
  • message that you want to send. This is a JSON message.

STEP 3: Run this program, and see your device UUID appear on the dashboard. Huzzzaaaaah! Thats it!

Arduino

This post takes you through the process of connecting the Arduino to the PubNub network and communicating to other devices via the dashboard.

The Arduino is a tool for making computers that can sense and control more of the physical world than your desktop computer. It's an open-source physical computing platform based on a simple microcontroller board, and a development environment for writing software for the board. Check out their website at Arduino for more information.

PubNub provides extensive documentation and support for Arduino. You can find the SDK for Arduino here.

STEP 1: Connect the Arduino to a computer using the USB port. Using an Ethernet shield, connect the Arduino board to the internet. Now download the Arduino IDE for your PC here . Full and comprehensive instructions for installing the software for Windows, Linux and Mac can be found at this site.

Once you have successfully installed the Arduino Software and, depending on your platform, USB drivers, you should now be able to upload the below program to the Arduino board.

STEP 2: Open a new sketch and paste the following code. 

#include <SPI.h>
#include <Ethernet.h>
#include "string.h"
#include "ardui_wrapper.h"

// Some Ethernet shields have a MAC address printed on a sticker on the 
shield;
// fill in that address here, or choose your own at random:
byte mac[] = { 0xDE, 0xAD, 0xBE, 0xEF, 0xFE, 0xED };

char pubkey[] = "demo";
char subkey[] = "demo";
char channel[] = "button-reply";
char uuid[] = "Arduino";

dashboard ard;

void do_something(String value){
          Serial.println("in the callback");
          Serial.println(value);
        }

void setup()
{
       
	Serial.begin(9600);
	Serial.println("Serial set up");

	while (!Ethernet.begin(mac)) {
		Serial.println("Ethernet setup error");
		delay(1000);
	}
	Serial.println("Ethernet set up");

        
	ard.dashinit( pubkey, subkey, uuid); 
	Serial.println("PubNub set up");
       
}

void loop()
{      
        String returnmessage;
	Ethernet.maintain();
        
        //Publish
	Serial.println("publishing a message");
        ard.dashsend(channel,"\"Hey There\"");
        

        //Subscribe
	Serial.println("waiting to receive a message");
        returnmessage = ard.dashconnect(channel);
        
        // callback function of sorts, to work with the received message
        do_something(returnmessage);
        
	Serial.println();
}

The above code lets you set up the following through the dashinit() method:

  • publish key using "pubkey"
  • subscribe key using "subkey"
  • UUID using "uuid"
  • channel you wish to subscribe and publish to using "channel"

STEP 3: Upload this program on your Arduino, and see your device UUID appear on the dashboard. Huzzzaaaaah! Thats it!

Visual Dashboard

PubCon allows any embedded device to talk to another. Whether its a PIC32 or an arduino board, all you need to do is to plug and start connecting. The dashboard just subscribes and publishes to the same channel as the devices. It also uses the presence API to detect the different devices connected to it.
